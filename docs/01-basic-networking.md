# Basic Networking

## Concepts
- IP Address: Unique identifier for devices in a network
- Subnet: Divides network into smaller segments
- Gateway: Path to other networks (usually router)
- DNS: Converts domain names into IP addresses

## Example
192.168.10.1/24
- Network: 192.168.10.0
- Host Range: 192.168.10.1 - 192.168.10.254
